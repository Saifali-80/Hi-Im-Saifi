# Interactive Resume Website

A modern, fast, accessible, and offline-capable professional resume website.

## Features
- **Performance First**: Optimized for speed, targeting <100KB gzipped bundle.
- **Accessibility**: WCAG 2.1 AA compliance with semantic HTML and keyboard support.
- **Offline Ready**: PWA support with Service Worker caching.
- **Dark Mode**: Respects system preferences and user choice.
- **SEO Optimized**: Includes Open Graph, Twitter Cards, and JSON-LD metadata.

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Modern features like `clamp()`, variables)
- Vanilla JavaScript ("use strict", performance-oriented)

## File Structure
- `index.html`: Main entry point.
- `css/`: Stylesheets.
- `js/`: Client-side logic.
- `assets/`: Images, fonts, and PDF resume.
- `sw.js`: Service worker for offline functionality.
- `docs/`: Project documentation (Constitution & Specification).
